### vi편집기 esc 안되는 현상

  Ctrl + c, shift + esc 모두 안되길래 찾아봤다

  Setting - Tools - Terminal - Override IDE shortcuts 옆의 Configure terminal keybindings

  검색창에 Switch Focus To Editor 항목이 escape로 되어있을 것  
  해당 키를 변경하거나 해제시켜주면 vi편집기에서 esc사용이 가능하다
